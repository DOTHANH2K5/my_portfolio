# My Portfolio

Welcome to my personal portfolio website! This is a modern, responsive portfolio template to showcase your projects, skills, and get in touch with potential clients or employers.

## 📁 Project Structure

```
portfolio/
│
├── index.html          # Main HTML file
├── style.css           # CSS styling
├── script.js           # JavaScript functionality
│
├── images/
│   ├── avatar.jpg      # Profile avatar image
│   └── project1.jpg    # Project showcase image
│
└── README.md           # This file
```

## 🎯 Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI**: Clean and professional design with smooth animations
- **Navigation**: Smooth scrolling navigation menu
- **Hero Section**: Eye-catching banner with call-to-action button
- **About Section**: Space to introduce yourself
- **Projects Showcase**: Grid layout for displaying your projects
- **Skills Section**: Display your technical skills
- **Contact Form**: Get in touch form for visitors
- **Footer**: Social media links and copyright information

## 🚀 Getting Started

1. **Clone or download** this repository
2. **Replace placeholder content** with your own information
3. **Add your images** to the `images/` folder
4. **Update the HTML** with your projects, skills, and about information
5. **Open `index.html`** in your browser to view

## 🎨 Customization

### Colors
Edit the color scheme in `style.css`:
- Primary gradient: `#667eea` to `#764ba2`
- Accent color: `#3498db`
- Background: `#2c3e50`

### Fonts
The default font is 'Segoe UI'. To change it, modify the `font-family` in `style.css`

### Images
Replace placeholder images in the `images/` folder:
- `avatar.jpg`: Your profile picture (recommended: 200x200px)
- `project1.jpg`: Project screenshot (recommended: 400x300px)
